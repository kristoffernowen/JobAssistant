# 2026-06-22 JobStream Raw JSON Logging

## Instruktion

Visa hur de råa JSON-objekten från JobStream ser ut och vilka datafält som finns.

- Lägg till loggningsfunktion som skriver till textfil i projektet.
- Det räcker att logga de tre första objekten.
- I `JobStreamClient` ska loggen visa råa JSON-objektet som kommer in före/vid deserialisering.
